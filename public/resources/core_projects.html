We are committed to producing free and open source software. Here you'll find the tooling we've built to run RbMU.

Core Projects

Much of our current work is RbMU-specific, but may still be of interest to folks running online courses. Active projects include:

  • university-web: Course administration and student management web app
  • mendibot: IRC bot which among other things posts chat transcripts to university-web
  • scheduler: Utility for scheduling meetings (MIT License)
  • puzzlenode: Quiz application inspired by Project Euler and the Internet Problem Solving Contest (IPSC)

Unless otherwise noted, all RbMU core projects are released under the GNU Affero GPL version 3
